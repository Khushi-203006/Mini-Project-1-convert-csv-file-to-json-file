# Python Modules Explanation

This repository contains a simple explanation and examples of commonly used Python modules in data processing and data engineering.

## Modules Covered
- glob – Used to retrieve file paths using pattern matching.
- os – Used for interacting with the operating system and handling file paths.
- json – Used for reading and writing JSON formatted data.
- pandas – Used for data analysis and working with CSV or structured data.

## Purpose
The goal of this project is to help beginners understand how these modules are used while working with files, schemas, and datasets in Python.

## Technologies Used
- Python
